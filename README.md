# 🎵 سورس ميوزك (نسخة 2026) 🚀

أسرع وأخف سورس ميوزك عربي مفتوح المصدر، مصمم ليعمل بكفاءة عالية على أنظمة **Ubuntu 22.04** وبأقل استهلاك للموارد (مثالي لرام 512MB).

---

## 🌟 المميزات (Features)
* **عربي بالكامل:** جميع الأوامر والردود باللغة العربية.
* **بدون رموز:** يدعم التشغيل المباشر (مثال: `شغل حاتم العراقي`) بدون الحاجة لنقطة أو سلاش.
* **اشتراك إجباري:** ميزة التحقق من الاشتراك في قناتك قبل التشغيل.
* **خفيف جداً:** مبني على مكتبات `Pyrogram` و `PyTgCalls` لضمان السرعة.
* **دعم يوتيوب:** تحميل وتشغيل فوري باستخدام `yt-dlp`.

---

## 🛠️ المتطلبات (Requirements)
* Python 3.10 أو أحدث.
* FFmpeg (لمعالجة الصوت).
* API_ID & API_HASH (من موقع my.telegram.org).
* BOT_TOKEN (من @BotFather).
* STRING_SESSION (جلسة الحساب المساعد).
